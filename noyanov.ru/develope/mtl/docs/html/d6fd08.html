RedundandEquationSystem

Matrix
   RedundandEquationSystem

Redundand Equation system Ax=~B. there A(mxn) is not square matrix (m>n) The equation system is resolved by least squares methods. The initial matrix system is resolved by leading into square matrix by multiplication both its parts to AT.
