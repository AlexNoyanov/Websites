SymmetricMatrix::Decomposite 
bool SymmetricMatrix::Decomposite(SquareMatrix<MatrixType>& T,Vector<int>& iscomplex)

Remarks

Decomposite square matrix into two triangle matrixes. This function is used for matrix inversion and resolution of equation system.

Implementation Notes

Step 1. Represents matrix A as two triangle matrixes T1 and T2 that are simmetric each other. iscomplex is array of flags to handle complex numbers
