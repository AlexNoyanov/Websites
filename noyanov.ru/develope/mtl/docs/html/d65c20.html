Matrix::Matrix 
Matrix::Matrix()
Matrix::Matrix(int rows,int columns,const MatrixType* initial_data=NULL)
Matrix::Matrix(const Matrix<MatrixType>& initial_matrix)

Parameters

rows

number of rows in matrix

columns

number of columns in matrix

initial_data

initial data for the storage (values are stored by rows)

initial_matrix

matrix with data to initiate

Remarks

Empty Default constructor for derived classes. It does not allocate memory for value storage. To alloc memory and make real matrix creation you have to call Matrix::ResetDimensions function with necessary number of rows and columns.

Desctiptions: Creates matrix with necessary number of rows/columns. Allocate memory to store matrix values and initialize them.
