SymmetricMatrix::operator+= 
SymmetricMatrix<MatrixType>& SymmetricMatrix::operator+=(const SymmetricMatrix<MatrixType>& other)

Return Value

A reference to the current object.

Implementation Notes

This sets the current object to the sum of the original value of this object and the value on the right.

Example

Matrix<double> a(3,3),b(3,3); a+=b;

To Matrix Summation Operator. Adds constant to this matrix.

Matrix<double> a(3,3); a+=3;
