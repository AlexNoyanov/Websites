Matrix Template Library

The Matrix Template Library is freeware C++ template class library to provide basic and enhanced matrix storages, matrix operation and functions laid on matrixes (linear equation system solving etc).

You can free download MTL sources. If you want to edit the sources please mail me.

Also MS Help and HTML documentation are available (documentation is generated from the sources).

If you want a information on the MTL or if you want a class, member function and so on, mail me. Your mail will serve to enhance the library only. I will not distribute your mail address anywhere.
